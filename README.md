### Cara publish (singkat)
1. Buat repo GitHub `nasar-17` (atau gunakan repo yang sudah ada).
2. Clone atau copy file dari folder ini ke repo lokal Anda.
3. Commit dan push, lalu aktifkan GitHub Pages di branch `main` (root).

Contoh perintah:

```bash
cd path/to/repo
cp -r "C:/Users/Pongo/Downloads/nasar-17-readme-prep/." .
git add .
git commit -m "Add profile README + interactive page"
git push origin main
```

Setelah Pages aktif, buka `https://nasar-17.github.io/` (jika repo bernama `nasar-17`) atau `https://USERNAME.github.io/nasar-17/`.
